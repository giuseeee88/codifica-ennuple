# Codifica N-uple
Questo progetto ha lo scopo di codificare le istruzioni e quindi codificare un interno programma scritto in Python (o scritto in altri linguaggi che in futuro probabilmente verranno aggiunti), al momento il progetto è all'inizio ed è in grado di codificare solo n-uple mediante la funzione di Pairing e i Numeri di Gödel.
